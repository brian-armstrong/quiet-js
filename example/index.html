Quiet.js

Transmit and receive data in the browser at 44.1kHz

Quiet.js transmits data from your browser with Web Audio. It is a binding to libquiet, compiled with emscripten. Quiet is powered by liquid dsp, by Joseph Gaeddert, and by libfec, by Phil Karn. Profile system built with libjansson.
