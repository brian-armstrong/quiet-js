disconnect()

Disconnect quiet.js from its microphone source This will disconnect quiet.js's microphone fully from all receivers This is useful to cause the browser to stop displaying the microphone icon Browser support is limited for disconnecting a single destination, so this call will disconnect all receivers. It is highly recommended to call this only after destroying any receivers.
